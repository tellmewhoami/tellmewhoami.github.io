数值计算方法

感悟：工作之后，很多理论的东西都还给原来的老师了。所以现在需要从头开始一点点啃原来的知识点，虽然会很难，但是请务必加油！在年前已经完成了线性代数的一部分学习，现在我需要进行数值计算方法的相关学习，以下是我的学习笔记。
网址：https://github.com/copilot/c/08d2ed67-7275-46ea-87f9-93bc8289ca37

数值计算中常用的方法（按“思想/用途”分类）

数值计算（Numerical Computation / Numerical Analysis）是研究“用计算机在有限精度下，用可操作的算法去近似求解数学问题”的学科与方法体系。它关注的不是写出解析解（精确公式），而是当解析解难以得到或代价太高时，如何通过离散化、迭代、逼近等手段，计算出足够准确、可控误差的近似解。

数值计算通常要同时考虑三件事：

  • 误差来源：模型/离散误差、截断误差、舍入误差（浮点数精度导致）。
  • 稳定性：小的输入扰动或舍入误差会不会被算法放大。
  • 效率：时间复杂度、空间复杂度、收敛速度（迭代法尤其重要）。

1) 近似替代法（逼近思想）

核心：用“更容易计算的对象”替代原问题中的“难对象”，并控制误差。常见形式：

  • 函数近似/插值：用多项式、样条（spline）等逼近函数
    • 拉格朗日插值、牛顿插值
    • 分段线性插值、三次样条插值
  • 多项式/级数逼近：
    • 泰勒展开（在某点附近用多项式近似）
    • 切比雪夫多项式逼近（常用于控制最大误差）
  • 数值积分（求面积/期望）：把积分用加权求和替代
    • 梯形法、辛普森法
    • 高斯求积
  • 数值微分：用差分替代导数
    • 前向差分、后向差分、中心差分（一般更准）

应用：当你要算 sin(x)、积分、导数、复杂函数值、物理/工程中的连续模型时，近似替代几乎无处不在。

2) 迭代法（逐步逼近解）

核心：把求解转化为“重复更新”，逐步逼近真实解，直到误差足够小。

  • 求方程根 f(x)=0：
    • 二分法（稳健但慢）
    • 牛顿法（快但依赖初值与导数，可能不收敛）
    • 割线法（不用导数，速度介于两者之间）
  • 解线性方程组 Ax=b（大规模时常用迭代）：
    • Jacobi、Gauss–Seidel、SOR
    • 共轭梯度（CG，适合对称正定）
    • GMRES（更一般）

关键点：收敛条件、停止准则（残差/相对误差）、迭代次数与计算代价。

3) 离散化方法（把连续问题变成离散问题）

核心：把微分方程、积分方程等连续模型，转成有限维的代数问题再求解。

  • 常微分方程 ODE：
    • 欧拉法、改进欧拉法（Heun）
    • Runge–Kutta（如经典四阶 RK4）
  • 偏微分方程 PDE：
    • 有限差分法（FDM）
    • 有限元法（FEM）
    • 有限体积法（FVM）

关键点：网格大小（步长）与误差阶数、稳定性（例如显式方法的稳定步长限制）。

4) 优化与最小二乘（把问题变成“最好”的那个）

核心：很多计算问题可以写成最小化误差/能量的形式。

  • 最小二乘拟合：线性回归、曲线拟合
  • 无约束优化：梯度下降、牛顿法、拟牛顿（BFGS）
  • 约束优化：拉格朗日乘子、内点法（更高级）

5) 线性代数数值方法（数值计算的“骨架”）

大量工程计算最终都落到矩阵运算上。

  • 矩阵分解：LU、QR、Cholesky、SVD
  • 特征值问题：幂迭代、QR 算法
  • 病态性与条件数：衡量问题对扰动敏感程度；决定你能“算多准”。

6) 随机/采样方法（当维度高或模型复杂）

  • 蒙特卡洛积分（高维积分常用）
  • 随机优化/采样：MCMC（更偏统计计算）

数值计算误差

1) 什么是“误差”

在数值计算里，**误差（error）**通常指“真实值”和“近似值/计算值”的差异。

设真实值为 (x)，近似值为 (\tilde x)，则

  • 绝对误差：[
    e_a = |\tilde x - x|
    ]
  • 相对误差：[
    e_r = \frac{|\tilde x - x|}{|x|}\quad(x\neq 0)
    ]
  • 百分相对误差：(100% \times e_r)

补充：有时也看“残差”（residual），它不是误差本身，但常用于估计误差。例如解方程 (f(x)=0) 得到近似解 (\tilde x)，残差为 (r=f(\tilde x))。

2) 数值计算误差的来源与分类（最常用的框架）

数值误差常被分成以下几类（对应来源也很清晰）：

A. 模型误差（Modeling error）

来源：真实世界问题被数学模型简化/假设导致偏差。
例子：忽略空气阻力、把非线性材料当作线性、用常系数近似变系数。

模型误差通常不属于“算法本身”的误差，但在工程中往往最大。

B. 数据/测量误差（Data / Measurement error）

来源：输入数据本身不精确（测量噪声、采样误差、记录精度）。
例子：传感器读数只有 0.01 精度；初始条件有噪声。

C. 截断误差（Truncation error）

来源：把无限过程用有限过程替代（截断级数、截断迭代、用差分近似导数）。
典型场景：

  • 泰勒展开只保留前几项
  • 数值积分把连续积分改成有限求和
  • ODE 用步长 (h) 的方法近似（误差随 (h) 变化）

截断误差常与“方法的阶”相关：例如某方法误差是 (O(h^2))，意味着步长减半，误差大约缩小到原来的 (1/4)。

D. 舍入误差（Round-off error，浮点误差）

来源：计算机用有限位数表示实数（IEEE 浮点），每次运算都可能产生舍入。
典型现象：

  • 吸收（absorption）：大数 + 很小的数，小数被“吃掉”
  • 灾难性消去（catastrophic cancellation）：两个很接近的数相减导致有效位数大量丢失
  • 误差累积/传播：大量运算后舍入误差积累并被放大

E. 算法误差/迭代误差（Iteration / algorithmic error）

来源：迭代法提前停止，或算法本身近似。
例子：牛顿法只迭代 5 次；线性方程组迭代法残差未充分小。

F. 离散化误差（Discretization error）

很多教材把它作为截断误差的一部分，但在微分方程数值解里常单独强调：
来源：连续问题（函数、导数、积分、PDE）→ 网格/节点上的离散问题。
例子：用网格间距 (h) 代替连续空间。

3) 误差的进一步常用“分类口径”

(1) 绝对误差 vs 相对误差

上面已给出。相对误差更能反映“精度占比”，但当 (x) 接近 0 时不稳定。

(2) 前向误差 vs 后向误差（重要）

  • 前向误差（forward error）：输出值离真实输出有多远
    [
    |\tilde y - y|
    ]
  • 后向误差（backward error）：等价于“输入数据改动多少，才能让算法输出变成精确解”
    例如：你算的是 (\tilde y = f(\tilde x))，能否解释为“(\tilde x = x + \delta x)”使得 (\tilde y = f(x+\delta x)) 恰好成立。

后向误差经常用来讨论数值稳定性：

  • 后向稳定：算法输出等价于对输入做了很小扰动后的精确结果（通常被认为是好算法）。

(3) 随机误差 vs 系统误差（更多用于数据/测量）

  • 系统误差：有偏差（总是偏大/偏小）
  • 随机误差：围绕真值随机波动

4) 什么是“有效数字”（significant digits）

有效数字用于描述一个近似值中“可信的数字位数”。

直观理解：近似值 (\tilde x) 中，从第一个非零数字开始，连续有多少位是可靠的。

更“数值分析式”的定义常用相对误差来刻画：
如果 (\tilde x) 近似 (x) 满足
[
\frac{|\tilde x-x|}{|x|} \le 0.5\times 10^{-t},
]
则说 (\tilde x) 至少有 (t) 位十进制有效数字。

例：若相对误差 (\le 5\times 10^{-4})，则有约 3 位有效数字（因为 (0.5\times 10^{-3}=5\times 10^{-4})）。

相关概念：

  • 正确位数：也常用类似方式定义（不同教材系数可能略有差异）。
  • 有效位数丢失：灾难性消去会显著降低有效数字。

5) 什么是“误差估计”（error estimation）

误差估计指：在真值 (x) 不可得的情况下，用可计算的信息去评估或界定误差的大小，典型目标是得到：

  • 误差上界（保证型）：( |\tilde x - x| \le \text{bound})
  • 误差近似量（经验型/渐近型）：( |\tilde x - x| \approx \text{estimate})

常见误差估计思路（非常核心）：

A) 先验误差估计（a priori）

在计算前，根据理论推导给出误差与参数（如步长 (h)、截断阶数 (n)）的关系。
例：梯形积分法误差 (\sim O(h^2))，可在选步长前预测精度等级。

B) 后验误差估计（a posteriori）

计算后，利用结果本身（或两次计算的差异）估计误差。常���方法：

  • 网格/步长加密对比：用 (h) 与 (h/2) 两个结果差来估误差（Richardson 外推思想）
  • 迭代差：迭代法中用 (|x_{k+1}-x_k|) 或残差来估计当前误差
  • 残差-误差关系：在某些问题里能把残差转换为误差界（例如线性方程组、某些 PDE 离散）

C) 用“残差”做误差估计（常见但需条件）

  • 对 (Ax=b)：残差 (r=b-A\tilde x)。在合适范数下，有
    [
    |\tilde x-x| \le |A^{-1}|\cdot |r|
    ]
    这给出一种从残差到误差的上界（��需要了解/估计 (|A^{-1}|)，与条件数有关）。
  • 对求根 (f(x)=0)：如果 (f) 在根附近光滑且 (f’(x)) 不太小，常有近似
    [
    |\tilde x - x| \approx \frac{|f(\tilde x)|}{|f’(\tilde x)|}
    ]
    这把残差 (f(\tilde x)) 转成误差量级（是近似估计，不一定严格上界）。

6) 小结（你可以用来背/做笔记）

  • 误差：真值与近似值差；常用绝对/相对误差表示。
  • 来源：模型、数据、截断/离散化、舍入、迭代停止等。
  • 有效数字：用相对误差阈值刻画近似值可信位数。
  • 误差估计：在未知真值时，通过理论（先验）或计算结果/残差/网格加密（后验）给出误差界或误差量级。

数值计算中需要注意的问题

在数值计算中，由于原始数据的误差，必然导致计算结果的差异，如果原始数据有扰动，但在误差传播过程中舍入误差能够控制，就称这种算法为数值稳定的；否者就是不稳定的。为了优化算法需要注意以下问题：

  1. 利用近似公式计算时，要注意收敛速度，讲究效率;
  2. 避免大数吃小数的情况；
  3. 先化简再计算，尽量减少运算步骤，避免误差累计；
  4. 应小心处理病态的数学问题；
  5. 设法控制误差的传播；
